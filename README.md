
# 💰 Personal Expenses Manager

A modern React application for tracking and managing monthly expenses.

## 🚀 Features

* Add expenses
* Delete expenses
* Monthly expense tracking
* Monthly history
* Total monthly spending
* Dashboard statistics
* Top spending category
* Monthly bar chart
* Category pie chart
* Local storage persistence
* Responsive mobile design

### Dashboard

Track your monthly spending with an intuitive dashboard.

### Monthly Analytics

* Monthly expense chart
* Category distribution chart
* Spending summary cards


## 🛠️ Built With

### Frontend

* React
* Vite
* Recharts
* React Select
* CSS3

### Storage

* Local Storage

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/linoyKaufman/personal-expenses-app.git
```

Navigate to the client folder:

```bash
cd personal-expenses-app/client
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
